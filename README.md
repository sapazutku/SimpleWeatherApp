
# Simple Weather App

This project is a simple weather iOS app with a MVC architecture. Developed for adesso internship case.


## Features

- Location based weather
- SpriteKit animations (Sunny, Rainy, Snowy, Drizzle, Cloudy)
- Search Cities
- MVC Architecture
